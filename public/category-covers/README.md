# Category Cover Images

Images in this folder are used only for category divider pages.

Use this naming convention:

```text
cat-alcohol.png
cat-cafe-te.png
cat-cervezas-maltas.png
cat-condimentos.png
cat-congelados.png
cat-conservas.png
cat-galletas.png
cat-golosinas.png
cat-harinas.png
cat-helados.png
cat-legumbres.png
cat-pastas.png
cat-pulpas.png
cat-refrescos.png
cat-refrigerados.png
cat-snaks.png
cat-varios.png
cat-zumos-lacteos.png
```

The paths are configured in `src/config/categories.js` through `coverImages[0]`.
If an image is missing, the divider page shows a placeholder.
